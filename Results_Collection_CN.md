# 当前计算结果

---

## 示例结果（p = 0.4）

对于无幸运的装备，假设：

- 每次尝试消耗：`1 个灵魂宝石`
- 目标等级：`+9`
- 成功概率：`p = 0.4`
- 失败概率：`1 - p = 0.6`
- 在等级 +1~+6 失败时，装备等级下降 1；在等级 +0、+7、+8 失败时，装备回到 +0。

从 `+0` 强化到 `+9` 的期望灵魂宝石消耗为：

```text
1298.056641 个灵魂宝石
```

（后面结构同英文，不再重复解释）

| 初始等级 | 强化到 +9 的期望灵魂宝石数 |
|---:|---:|
| +0 | 1298.056641 |
| +1 | 1295.556641 |
| +2 | 1289.306641 |
| +3 | 1277.431641 |
| +4 | 1257.119141 |
| +5 | 1224.150391 |
| +6 | 1172.197266 |
| +7 | 1091.767578 |
