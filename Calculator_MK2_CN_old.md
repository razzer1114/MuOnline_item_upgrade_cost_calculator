## 🎯 问题描述

> 在任意成功率和宝石价格条件下，从 +0 强化到 +9 的最优策略是什么？

本项目构建一个**通用优化模型**，能够：

- 适配任意成功率
- 适配任意价格体系
- 适用于不同服务器经济环境
- 自动输出最优策略

---

## 🧠 建模方法

### Bellman 动态规划

强化过程可表示为：

$$
V(i) = \min_{a \in A_i} \left[ c(i,a) + \sum_j P(j|i,a) V(j) \right]
$$

含义：

- 当前成本 + 未来最优成本
- 自动选择最优策略

---

### 吸收马尔可夫链

对于固定策略：

- 状态 = 装备等级
- +9 = 吸收态
- 强化 = 状态转移

期望成本：

$$
E = (I - Q)^{-1} c
$$

---

### 实际求解方法

策略数量：

```
2^6 = 64
```

因此直接：

```
穷举策略 + 马尔可夫计算
```

## 📊 示例结果（真实经济数据）

**核心结论：模型可直接结合真实市场价格给出最优强化策略。**

2026年4月28日数据：

- 1元 = 7.14 灵魂
- 1元 = 1.35 祝福

因此：

```
1祝福 ≈ 5.29灵魂
```

当：

```
p_soul = 0.5
```

最优策略：

| 等级 | 宝石 |
|---|---|
| +0→+1 | 灵魂 |
| +1→+2 | 灵魂 |
| +2→+3 | 祝福 |
| +3→+4 | 祝福 |
| +4→+5 | 祝福 |
| +5→+6 | 祝福 |
| +6→+7 | 灵魂 |
| +7→+8 | 灵魂 |
| +8→+9 | 灵魂 |

即：

```
SSBBBBSSS
```

  - +0：原地
  - +1~+6：退一级
  - +7,+8：回到+0
