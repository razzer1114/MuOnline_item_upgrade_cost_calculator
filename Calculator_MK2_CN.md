## 🎯 问题描述

> 在同时存在祝福宝石与灵魂宝石的情况下，从 +0 强化到 +9 的最优策略是什么？期望成本是多少？

---

## 🧠 建模思路

我们将强化过程建模为一个**带决策的吸收型马尔可夫链系统**，其中：

- 每个装备等级对应一个**状态**
- 目标等级（+9）是**吸收态**
- 每次强化是一次**状态转移**
- 在部分等级，玩家可以选择不同资源（动作）

---

## ⚙️ 策略设计

### 决策范围

- 等级 `+0 → +6`：
  - 可选择使用：
    - 祝福宝石（100%成功，成本较高）
    - 灵魂宝石（概率成功，成本较低）

- 等级 `+6 → +9`：
  - 必须使用灵魂宝石（无选择）

---

### 策略空间

每个决策点有两种选择：

```
祝福 / 灵魂
```

总策略数量：

```
2^6 = 64 种策略
```

策略可以表示为字符串：

```
例如：BBBSSSSSS
```

---

## 📊 示例结果

假设：

- 祝福成本：`5`
- 灵魂成本：`1`
- 成功概率：`p = 0.6`
- 失败概率：`1 - p = 0.4`

模型对全部 64 种策略进行评估，并得到：

```
最优策略：
SSSSSSSSS
```

期望总成本：

```
约 7.96（简化模型示例）
```

## 🔁 状态转移规则

设：

- `p` 为灵魂成功率
- `1-p` 为失败概率

### 祝福宝石

```
成功率 = 100%
必然升级
```

---

### 灵魂宝石

| 当前等级 | 成功 | 失败 |
|---|---|---|
| +0 | +1 | +0 |
| +1~+6 | +i+1 | i-1 |
| +7, +8 | +i+1 | +0 |
| +9 | +9 | +9 |

## 🧮 吸收马尔可夫链表示

$$
P =
\begin{bmatrix}
Q & R \\
0 & I
\end{bmatrix}
$$

其中：

- Q：暂态转移矩阵
- R：吸收到终态的转移矩阵

---

## 💰 期望成本计算

定义成本向量：

$$
c_i =
\begin{cases}
c_b, & 使用祝福\\
c_s, & 使用灵魂
\end{cases}
$$

期望成本：

$$
E = (I - Q)^{-1} c
$$

---

## 🔍 策略评估流程

对每个策略：

1. 构建 Q 矩阵
2. 构建成本向量 c
3. 计算：

$$
E = (I - Q)^{-1} c
$$

4. 提取：

```
E_0（从 +0 开始的期望成本）
```
